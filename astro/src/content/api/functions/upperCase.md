# Function: upperCase()

> **upperCase**(`str`): `string`

Converts a string to upper case

## Parameters

• **str**: `string`

string to convert

## Returns

`string`

the string in upper case

## Example

```ts
upperCase('foo') // returns 'FOO'
upperCase('FOO') // returns 'FOO'
upperCase('123') // returns '123'
upperCase('') // returns ''
upperCase('foo-bar') // returns 'FOO-BAR'
```
